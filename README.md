# Saycode
<img width="1383" height="758" alt="image" src="https://github.com/user-attachments/assets/a79461cb-66ec-4017-aa23-9ea80c406777" />
Desenvolvi um site com integração de uma API de IA para treinar requisições de API e aprofundar minha prática em JavaScript. O sistema envia código do front end para o back end, processa a análise usando IA e retorna o resultado em tempo real na interface.

Aprendi a fazer chamadas HTTP, lidar com respostas assíncronas e tratar erros de conexão. Trabalhei com variáveis de ambiente para proteger tokens de API e entendi a diferença entre rodar um projeto em localhost e em produção. Também organizei melhor meu código, separando HTML, CSS, JavaScript e back end.

Esse projeto foi meu primeiro teste prático com inteligência artificial. Durante o desenvolvimento, aprendi versionamento com Git, uso do GitHub, resolução de conflitos, regras de segurança do repositório e deploy do site. Estou aberto a receber comentários, sugestões e ajuda para evoluir o projeto. Pretendo implementar melhorias futuras, refatorar o código e adicionar novas funcionalidades conforme continuo aprendendo.
